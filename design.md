# 服薬管理アプリ デザイン仕様書

## アプリ概要
毎日の服薬を簡単に記録し、飲み忘れを防ぐためのモバイルアプリ。シンプルで直感的な操作で、高齢者でも使いやすいUIを目指す。

## 画面構成

### 1. ホーム画面 (Home)
**主要コンテンツ:**
- 今日の服薬スケジュール一覧（最大3回分）
- 各服薬時刻の状態表示（未服薬/服薬済み/飲み忘れアラート中）
- ワンタップで服薬記録できる大きなボタン
- 次の服薬時刻までのカウントダウン表示

**機能:**
- 服薬完了ボタンをタップすると即座に記録
- 飲み忘れ状態の場合は赤色で強調表示
- 全ての服薬が完了したら完了メッセージを表示

**レイアウト:**
- カード形式で各服薬時刻を表示
- 時刻の大きな表示と服薬完了ボタンを配置
- 下部にタブバー（ホーム/カレンダー/設定）

### 2. カレンダー画面 (Calendar)
**主要コンテンツ:**
- 月間カレンダービュー
- 各日付に服薬完了率を表示（例: 3/3, 2/3など）
- 完全達成日は緑色、未達成日は黄色/赤色でマーク

**機能:**
- 日付をタップすると、その日の詳細な服薬記録を表示
- 過去の記録を一覧で確認可能
- 月の切り替え機能

**レイアウト:**
- 上部に月の選択UI
- カレンダーグリッド表示
- 選択した日の詳細を下部に表示

### 3. 設定画面 (Settings)
**主要コンテンツ:**
- 服薬時刻の設定（最大3回）
- 各時刻のオン/オフ切り替え
- 通知設定の確認

**機能:**
- 時刻ピッカーで服薬時刻を設定
- 不要な時刻はオフにできる
- 設定変更時に通知スケジュールを自動更新

**レイアウト:**
- リスト形式で各服薬時刻を表示
- 時刻ごとに編集ボタンとトグルスイッチ
- 保存ボタンで設定を確定

## 主要ユーザーフロー

### フロー1: 服薬記録
1. ホーム画面を開く
2. 該当する服薬時刻のカードを確認
3. 「服薬完了」ボタンをタップ
4. 即座に記録され、カードの状態が「完了」に変更
5. ハプティックフィードバックで完了を通知

### フロー2: 服薬時刻の設定
1. 設定画面を開く
2. 「服薬時刻1」をタップ
3. 時刻ピッカーで時刻を選択（例: 08:00）
4. 保存ボタンをタップ
5. ホーム画面に反映され、通知スケジュールが更新される

### フロー3: 履歴確認
1. カレンダー画面を開く
2. 確認したい日付をタップ
3. その日の服薬記録詳細が表示される
4. 完了/未完了の状態を確認

### フロー4: 飲み忘れアラート
1. 設定時刻に通知が届く
2. 1時間経過しても記録がない場合、飲み忘れアラートが発動
3. 30分間隔で繰り返し通知
4. アプリを開いて服薬完了ボタンをタップするまで継続

## デザイン原則

1. **シンプルさ優先**: 高齢者でも迷わず使えるよう、1画面1機能を基本とする
2. **大きなタップ領域**: ボタンは最低44x44ptのタップ領域を確保
3. **明確なフィードバック**: 操作後は必ずハプティックフィードバックと視覚的変化を提供
4. **iOS HIG準拠**: ネイティブiOSアプリと同様の操作感を実現
5. **縦持ち片手操作**: 全ての主要機能は画面下半分で操作可能に配置
